# Jupyter-DataScience-OS-Projects
This repository contains a collection of interactive programs developed in Jupyter Notebook, focusing on three core areas:

Exploratory Data Analysis (EDA): Cleaning, transforming, and analyzing datasets using the Pandas and NumPy libraries to find hidden patterns.

Data Visualization: Creating insightful charts and graphs (Line plots, Scatter plots, Histograms) using Matplotlib and Seaborn.

Operating Systems (OS): Implementations of core OS concepts including Process Scheduling algorithms (FCFS, SJF) and memory management simulations.

Environment: Python 3, Jupyter Notebook
# Jupyter Notebook Projects: OS & Data Science

This repository contains a collection of Python programs implemented in **Jupyter Notebook**. These projects cover core concepts of Operating Systems and foundational techniques in Data Science.

## 📁 Repository Structure

*   **Exploratory Data Analysis (EDA):** Using `Pandas` to clean, manipulate, and analyze structured data.
*   **Data Visualization:** Using `Matplotlib` and `Seaborn` to create interactive and static plots.
*   **Operating Systems (OS):** Practical implementation of system-level algorithms in Python.

---

## 🚀 Projects Included

### 1. Exploratory Data Analysis (EDA)
- **Goal:** Analyzing datasets to summarize their main characteristics.
- **Key Features:** Handling missing values, data grouping, and statistical summaries using `df.describe()`.

### 2. Data Visualization
- **Goal:** Transforming raw data into visual insights.
- **Visuals:** Sine/Cosine waves, bar charts for sales data, and distribution histograms.

### 3. Operating Systems Algorithms
- **Goal:** Simulating how an OS manages processes and resources.
- **Implemented:** 
    - First-Come, First-Served (FCFS) Scheduling
    - Shortest Job First (SJF) Scheduling
    - Memory Management simulations

---

## 🛠️ Tools & Technologies
- **Language:** Python 3
- **Environment:** Jupyter Notebook / Google Colab
- **Libraries:** 
    - `NumPy` (Numerical computing)
    - `Pandas` (Data manipulation)
    - `Matplotlib` (Plotting and Viz)
